# Testing Documentation (Portfolio Summary)

This project includes multi-level testing documentation that connects expected behaviors to verifiable checks.

## Test plan
- Scope, test levels, entry/exit criteria, and reporting structure.

## Integration testing
- UI flows and feature integration checks (examples: settings persistence, save/load flows, parental controls).

## System testing
- End-to-end verification of core gameplay loop and user-facing requirements.

## Validation testing
- User-story level confirmation against requirements, with pass/fail outcomes and remarks.

Source artifacts are available under `source_pdfs/testing/`.
